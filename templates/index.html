YouTube Downloader API

A powerful REST API for downloading YouTube videos with multiple format options and comprehensive metadata extraction.

Multiple Formats

MP4 from 144p to 1080p

Audio Only

MP3 128kbps - 320kbps

Rich Metadata

Title, views, duration, etc.

API Documentation

POST /api/video-info

Description: Get comprehensive video information and available formats.

Request Body:
{ "url": "https://www.youtube.com/watch?v=VIDEO_ID" }
Response:
{ "success": true, "data": { "title": "Video Title", "channel": "Channel Name", "duration": 180, "view_count": 1000000, "like_count": 50000, "thumbnail": "https://...", "formats": { "video_formats": { "720p": [{"format_id": "...", "ext": "mp4", ...}], "1080p": [{"format_id": "...", "ext": "mp4", ...}] }, "audio_formats": { "128kbps": [{"format_id": "...", "ext": "mp3", ...}], "192kbps": [{"format_id": "...", "ext": "mp3", ...}] } } } }
POST /api/download

Description: Download video or audio file directly.

Request Body:
{ "url": "https://www.youtube.com/watch?v=VIDEO_ID", "quality": "720p", // optional: 144p, 240p, 360p, 480p, 720p, 1080p "audio_only": false, // optional: true for audio only "format_id": "FORMAT_ID" // optional: specific format ID }
Response:

Returns the file as a download attachment.

POST /api/download-link

Description: Get direct download URL for video or audio.

Request Body:
{ "url": "https://www.youtube.com/watch?v=VIDEO_ID", "quality": "720p", // optional "audio_only": false // optional }
Response:
{ "success": true, "download_url": "https://...", "filename": "video_title.mp4", "filesize": 15728640, "expires_in": "6 hours" }
POST /api/create-preview-link

Description: Create a preview page link where users can watch the video first and then decide to download.

Request Body:
{ "url": "https://www.youtube.com/watch?v=VIDEO_ID" }
Response:
{ "success": true, "preview_url": "https://your-domain.com/video/VIDEO_ID", "video_id": "VIDEO_ID", "original_url": "https://www.youtube.com/watch?v=VIDEO_ID" }
Preview Page Features:
  • Embedded YouTube video player
  • Video metadata (title, channel, views, likes)
  • Multiple download options with different qualities
  • Direct download buttons and link generation
